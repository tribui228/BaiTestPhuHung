# Project Title

## Giới thiệu

Đây là một dự án ví dụ sử dụng .NET 8 và Angular 18. Dự án bao gồm một ứng dụng web với các tính năng quản lý sản phẩm, bao gồm thêm, sửa, xóa và tìm kiếm sản phẩm.

## Công Nghệ Sử Dụng

- **Backend**: .NET 8
- **Frontend**: Angular 18
- **Cơ sở dữ liệu**: SQL Server
